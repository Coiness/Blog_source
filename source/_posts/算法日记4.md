[LeetCode指路](https://leetcode.cn/problems/max-consecutive-ones/description/) 难度：简单

给定一个二进制数组 `nums` ， 计算其中最大连续 `1` 的个数。

<!--more-->

#### 题解

##### 方法一：一次遍历

如题，通过遍历得出答案。

```cpp
int ans = 0;
int temp = 0;
for(int i = 0; i < nums.size(); i++){
    if(nums[i] == 1){
        if(++temp > ans){
            ans = temp;
        }
    }
    else{
        temp = 0;
}
}
```

挺简单的，很多题解思路都大差不差。
