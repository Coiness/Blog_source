[LeetCode指路]([https://leetcode.cn/problems/plus-one/description/) 难度：简单

给定一个由 **整数** 组成的 **非空** 数组所表示的非负整数，在该数的基础上加一。

最高位数字存放在数组的首位， 数组中每个元素只存储**单个**数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

<!--more-->

#### 题解

该题比较简单

需要注意的点是，当首位到末位都是9时，加一会使数字的位数增加，需要多加一位。

（如：9 9 9 9 +1 = 1 0 0 0 0）

##### 方法1：找出最长的后缀9

对数组digits进行一次逆序遍历，找出第一个不为9的数字，将其加1并后续的所有元素置零

```cpp
for(int i = digits.size()-1;i >= 0;i--){
    if(digits[i] != 9){
        ++digits[i];
        for(int j = i + 1;j < digits.size();j++){
            digit[j] = 0;
        }
        return digits;
     }
}

vector<int> ans(digits.size()+1);
ans[0] = 1;
return ans; 
```
